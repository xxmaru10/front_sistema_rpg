1. **AtivaГғВ§ГғВЈo**: O usuГғВЎrio trigga uma aГғВ§ГғВЈo na UI (ex: "Causar Dano").

4. **Relay**: O backend carimba o evento e rebate para todos os outros clientes na sala via WebSocket.
